## **前提条件**

- 已开通阿里云百炼服务，并已获取API-KEY：[获取API Key](https://help.aliyun.com/zh/model-studio/developer-reference/get-api-key)。

## 功能特点

- 只要你有apikey，就可以自己使用
- 有完整的UI不用关心具体实现，关心可以自行看源码
